🛒 AI-Based Wholesale Customer Segmentation
📌 Project Overview

This project focuses on Customer Segmentation using Unsupervised Machine Learning techniques applied to the Wholesale Customers Dataset.

The goal is to identify groups of customers based on their purchasing behavior across different product categories. By applying clustering algorithms, businesses can better understand customer patterns and make data-driven decisions.

This project was developed as part of an AI training program.

🧠 Problem Statement

Wholesale businesses serve customers with different buying patterns. Understanding these patterns helps organizations to:

Identify high-value customer groups

Improve marketing strategies

Optimize inventory management

Support data-driven business decisions

Clustering techniques are used to group customers according to their annual spending behavior.

📊 Dataset Information

Dataset: Wholesale Customers Dataset

Source: Kaggle

Type: Customer purchasing data

Features in Dataset

Fresh

Milk

Grocery

Frozen

Detergents_Paper

Delicassen

Each row represents a customer's annual spending in different product categories.

⚙️ Machine Learning Techniques Used
1️⃣ K-Means Clustering

Applied K-Means algorithm to segment customers

Used the Elbow Method to determine the optimal number of clusters

Groups customers based on spending similarity

2️⃣ Hierarchical Clustering

Implemented Agglomerative Hierarchical Clustering

Used Dendrogram visualization to understand cluster formation

Helps analyze relationships between different customer groups

📈 Key Features of the Project

Data preprocessing and cleaning
